# Запуск проекта Vite + React + TypeScript

## Установка:

1. Открыть консоль

2. Клонировать репозиторий (или скачать ZIP)

```
git clone https://github.com/drapkinamaria/vk-frontend.git
```

3. Перейти в проект

```
cd vk-frontend
```

4. Установить зависимости

```
npm install
```

5. Запуск

```
npm run dev
```
